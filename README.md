# Simple Python for Autodesk Flame

A collection of Python utility scripts for [Autodesk Flame](https://www.autodesk.com/products/flame/overview) — a professional visual effects and finishing system.

## Scripts

| Script | Description |
|--------|-------------|
| `_BB_append_startframe_to_name` | Appends start frame number to clip name |
| `_BB_black_head_and_tail` | Adds black frames to head and tail of clips |
| `_BB_cache_motion_vectors` | Caches motion vectors for clips |
| `_BB_clip_path_to_clipboard_for_slack` | Copies clip path to clipboard for sharing |
| `_BB_color_timewarp_shots` | Applies color timewarp to shots |
| `_BB_create_marker` | Creates markers on clips |
| `_BB_create_reel_group` | Creates and organizes reel groups |
| `_BB_delete_empty_tracks` | Removes empty tracks from timeline |
| `_BB_freeze_frame_mux` | Creates freeze frame mux |
| `_BB_merge_offline` | Merges offline media |
| `_BB_open_photoshop` | Opens selected clip in Photoshop |
| `_BB_quick_posting_mp4` | Quick MP4 export for posting/review |
| `_BB_ratio_bug_fix` | Fixes aspect ratio issues |
| `_BB_remux_to_mp4` | Remuxes clips to MP4 format |
| `_BB_rename_9_and_12` | Renames clips for 9:16 and 12:12 formats |
| `_BB_set_in_out` | Sets in/out points on clips |
| `_BB_start_project` | Project startup automation |
| `_BB_surround_sound_mute_and_unmute` | Mutes/unmutes surround sound channels |
| `_BB_update_slate_date` | Updates date on slate |

## Usage

Copy scripts into your Flame Python scripts directory. Scripts are designed to be run from within Flame's Python scripting environment.
